# 旅行紹介ミニ Web サイト（React + Vite）

このリポジトリは、**React + Vite** を使用して作成した  
シンプルな旅行紹介用のミニ Web サイトです。  
React の基礎的なコンポーネント構成、props、イベント処理、  
そして Vite を利用した高速な開発環境の学習を目的として作成しました。

---

## プロジェクト概要

このアプリでは、旅行スポットをカード形式で表示し、  
簡単な UI レイアウトやコンポーネント分割の練習ができます。

実装している内容：

- React コンポーネントの基本構造
- Props を使ったデータ表示
- 画像を用いた観光地カードレイアウト
- シンプルな CSS スタイリング
- Vite を使用した高速開発体験

個人学習や React の入門に最適な軽量プロジェクトです。

---

## 使用技術

- **React 18**
- **Vite**
- **TypeScript (ES6+)**
- **CSS / module.css**
- Node / npm
- Git / GitHub
